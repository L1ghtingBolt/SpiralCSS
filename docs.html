Other classes

SpiralCSS has some more classes:
w-{num 0-30} Width
p-{num 0-30} Padding
m-{num 0-30} Margin
h-{num 0-30} Height
You can use (l)eft, (r)ight, (b)ottom adn (t)op for any of these. For example:
ml-30 (margin-left: 750px)
There's the font-{num 0-30}, too. It works for the font size. Another classes are:
flex, block, inline, inlineblock, italic, bold, normal, underline, linethrough, dec-none
